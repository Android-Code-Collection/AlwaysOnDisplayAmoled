Få en alltid på display för din telefon eller surfplatta.
Håll din skärm på hela tiden.
Den första alltid på appen med öppen källkod!

Tanken är att förse användarna med information om, tid, datum, meddelanden och mer och allt utan att behöva röra telefonen. Bara genom att titta på den.

Detta görs möjligt tack vare AMOLED skärmar. Mestadels av skärmen förblir svart med undantag för ett fåtal pixlar.

★Anmärkningsvärda funktioner
• Alltid på skärm
• Meddelanden - Visa meddelanden utan att röra din enhet
• Automatiska regler - bevara batteriet med hjälp av fördefinierade regler
• Automatisk rörelse - undvika AMOLED inbränning
• Auto nattläge - tona ned skärmen automatiskt i mörk miljö
• Ficka läge - låsa enheten när du lämnar den i fickan för att spara batteri
Dubbeltryck för att väcka + Dra upp för att väcka + volymknapparna för att vakna + bakåtknappen för att väcka
Force orientering - ställa in önskad skärmorientering
• Anpassning - ändra textfärg, textstorlek, ljusstyrka och mer
• FRI OCH ÖPPEN KÄLLKOD!

Innan du använder Vänligen aktivera "dra över appar" tillstånd för denna app.

Appen är i BETA, den har fortfarande buggar och saknar funktioner.

Behörigheter
<b>Kamera tillstånd</b>
Appen behöver kamera tillstånd för att känna igen kamerastart och stänga av "alltid på" skärmen.
<b>Telefon tillstånd</b>
Appen behöver telefon tillåtelse att känna igen inkommande samtal, avvisa alltid på skärmen och visa inkommande samtalsskärmen.
<b>Tillstånd för att ändra system inställningar</b>
Appen behöver behörighet att ändra inställningar för att ändra ljusstyrkan på låsskärmen.
