Få en "always on" display för din telefon eller surfplatta.
Håll din skärm på hela tiden.
Den första "always on" appen med öppen källkod!

Tanken är att förse användarna med information om, tid, datum, meddelanden och mer utan att behöva röra telefonen. Bara genom att titta på den.

Detta görs möjligt tack vare AMOLED skärmar. Det mesta av skärmen förblir svart med undantag för ett fåtal pixlar.

★Anmärkningsvärda funktioner
•    Always on skärm
•    Meddelanden - Visa meddelanden utan att röra din enhet
•    Automatiska regler - Spara ström med hjälp av förbestämda regler
•    Automatisk rörelse - Undvik AMOLED inbränning
•    Automatiskt nattläge - Tona ned skärmen automatiskt i mörk miljö
• Fickläge - Spara ström genom att låsa enheten när du stoppar ner den i fickan
• Dubbeltryck för att väcka + Dra upp för att väcka + volymknapparna för att väcka + bakåtknappen för att väcka
•    Skärmorientering - ställ in önskad skärmorientering
•    Anpassning - Ändra textfärg, textstorlek, ljusstyrka och mer
•    FRI OCH ÖPPEN KÄLLKOD!

Innan du använder vänligen aktivera behörighet för att "dra över appar" i denna app.

Denna app är i BETA, den har fortfarande buggar och saknar funktioner.

Behörigheter
Behörighet för kamera
Appen behöver behörighet för kameran för att känna igen kamerastart för att stänga av "always on" skärmen.
Telefon behörighet
Appen behöver behörighet för samtal för att känna igen inkommande samtal, avvisa 'always on" skärmen och visa inkommande samtalsskärmen.
Behörighet för att ändra system inställningar
Appen behöver behörighet att ändra inställningar för att ändra ljusstyrkan på låsskärmen.
